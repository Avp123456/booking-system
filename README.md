# Booking System

A RESTful Booking System built with Spring Boot and MySQL.  
Allows users to book resources (rooms, vehicles, equipment) with **role-based access control (ADMIN/USER)** and **JWT authentication**.

---

## Tech Stack

- Java 17+
- Spring Boot 3.x
- Spring Security (JWT)
- MySQL
- Lombok (optional)
- Maven
